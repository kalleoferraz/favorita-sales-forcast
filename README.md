# Favorita Sales Forecasting

End-to-end time series forecasting project · Corporación Favorita

## Project Structure

```
favorita-sales-forcast/
├── notebooks/
│   ├── 01_eda_and_cleaning.ipynb       # EDA & data preparation
│   ├── 02_feature_engineering.ipynb    # Feature matrix for ML models
│   ├── 03_statistical_models.ipynb     # SARIMA, Holt-Winters, Prophet
│   └── 04_ml_models.ipynb              # RF, XGBoost, HyperOpt, MLflow
├── data/
│   ├── timeseries.csv                  # Raw daily sales (place here)
│   ├── oil.csv                         # Oil prices
│   ├── holidays.csv                    # Ecuador public holidays
│   └── stores.csv                      # Store metadata
├── models/                             # Generated by notebooks
├── app.py                              # Streamlit app
├── requirements.txt
└── README.md
```

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run notebooks in order (1 → 4)

# 3. Launch the Streamlit app
streamlit run app.py
```

## Business Problem
Shop owners need to know expected demand for the next 7–30 days to
avoid stockouts (lost sales) and overstock (wasted perishables).

## Champion Model
Selected by lowest RMSE on the 90-day held-out test set (Jan–Mar 2014).
